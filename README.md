#Open-CV-Volume-Controller#
A real-time gesture-based control system that maps hand distances to Linux audio controls.

Project Overview
This project leverages computer vision to provide a hands-free, intuitive way to interact with system audio settings. By using hand-tracking technology, the system calculates the distance between specific landmarks on the hand and maps this value to proportionally control the volume on a Linux-based system.

Tech Stack
Language: Python
Computer Vision: OpenCV
Hand Tracking: MediaPipe
Operating System: Linux

Key Features
Real-time Interaction: Uses a webcam to detect hand gestures in real-time.

Proportional Control: Maps precise finger positioning to volume levels.

Linux Integration: Designed for Linux audio control interfaces.
